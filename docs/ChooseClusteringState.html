ChooseClusteringState

ChooseClusteringState

This step chooses between the k-means and SNN graph clusterings. We added this step to preserve the cache for each clustering step - specifically, each clustering does not need to be recomputed when a user changes their choice.

Methods not documented here are not part of the stable API and should not be used by applications.
