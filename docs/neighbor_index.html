neighbor_index

This step assembles the neighbor search indices from the PCs (see pca) in preparation for nearest neighbor searches in downstream steps. It wraps the buildNeighborSearchIndex function from scran.js.

The parameters in runAnalysis should be an object containing:

  • approximate: boolean indicating whether to create an approximate search index.

Calling the results() method for the relevant state instance will return an empty object.

Methods not documented here are not part of the stable API and should not be used by applications.
